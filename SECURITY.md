# セキュリティポリシー

## 脆弱性の報告

個人開発の小さな PWA ですが、脆弱性の報告は歓迎します。

公開の Issue ではなく、GitHub の **[Security Advisories（Private vulnerability reporting）](https://github.com/hazymoon/osanpo-bingo/security/advisories/new)** からご連絡ください。内容を確認し、修正方針を返信します。

## 対象範囲

- このリポジトリのソースコードおよびビルド成果物（静的 PWA）。
- バックエンドを持たず、写真などのデータは端末ローカルのみで完結する設計のため、サーバ側の脆弱性は対象外です。

## サポート対象

最新の `main` ブランチのみを対象とします。
